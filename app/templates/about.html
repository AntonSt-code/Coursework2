Про наш проєкт

Електронна бібліотека — це сучасна платформа для доступу до великої колекції електронних книг української та світової літератури.

Ми створили цей проєкт, щоб зробити читання доступним для всіх. Незалежно від того, де ви знаходитесь — вдома, на роботі чи в дорозі — ви завжди маєте доступ до ваших улюблених книг.

Наша місія — популяризація читання та збереження літературної спадщини для майбутніх поколінь.

Завантажень

Можливості платформи

Зручний пошук

Знайдіть потрібну книгу за назвою, автором, жанром або ISBN. Фільтри та сортування допоможуть швидко знайти те, що ви шукаєте.

Різні формати

Завантажуйте книги в форматах PDF, EPUB, MOBI та FB2. Читайте на комп'ютері, планшеті або електронній книзі.

Обране

Додавайте улюблені книги до обраного для швидкого доступу. Створюйте власну колекцію книг для читання.

Рейтинги та відгуки

Оцінюйте прочитані книги та залишайте відгуки. Допомагайте іншим читачам зробити правильний вибір.

Безпека

Ваші дані захищені сучасними методами шифрування. Ми піклуємося про вашу приватність.

Доступність

Responsive дизайн дозволяє комфортно користуватись сайтом на будь-якому пристрої — від телефону до комп'ютера.

Технології, що використовуються

Backend
  • Python 3.10+
  • Flask 3.0 (веб-фреймворк)
  • SQLAlchemy (ORM)
  • Flask-Login (аутентифікація)
  • Flask-WTF (форми)
Frontend
  • HTML5, CSS3
  • Bootstrap 5
  • JavaScript
  • Jinja2 Templates
  • Bootstrap Icons
База даних
  • MySQL 8.0+
  • Віддалений сервер
  • Flask-Migrate (міграції)
Безпека
  • Хешування паролів (Werkzeug)
  • CSRF-захист
  • SQL Injection захист
  • XSS захист
